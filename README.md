# Elementum — Strategic Design Agency

Elementum is a premium, modern, and highly interactive landing page for a strategic design and branding agency. It showcases the agency's team, philosophy, services, and creative progress.

## ✨ Features

- **Premium Aesthetics:** Sleek dark-mode and light-mode typography, custom HSL-tailored colors, and smooth gradients.
- **Dynamic Floating Avatar Arc:** An interactive visual display of team members that groups them in aesthetic pairs with smooth floating animations and pause-on-hover actions.
- **Responsive Layout:** Optimized for desktop, tablet, and mobile devices (utilizing a responsive flexbox fallback on smaller screens).
- **Creative Sections:**
  - **Hero Section:** Introduction with highlighted headings and the dynamic team avatar arc.
  - **Tomorrow Section:** Spotlights the agency's approach to designing the future.
  - **Progress Section:** Showcases key performance indicators, milestones, and status metrics.
  - **Testimonial Section:** Slider or grid featuring words from key clients and partners.
  - **Footer:** Clean layout with quick links, social media integration, and copyright info.

## 🛠️ Technology Stack

- **Framework:** React 19
- **Build Tool:** Vite 8 (with Hot Module Replacement)
- **Styling:** Vanilla CSS (Custom design system and theme tokens)
- **Linter:** Oxlint (for super-fast analysis)

## 🚀 Getting Started

Follow these steps to run the project locally on your machine:

### 1. Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed (v18+ recommended).

### 2. Installation

Install all the required dependencies using npm:

```bash
npm install
```

### 3. Run Development Server

Start the local development server:

```bash
npm run dev
```

The app will start at `http://localhost:5173/`. Open it in your browser to view the page.

### 4. Build for Production

To compile and optimize the assets for production:

```bash
npm run build
```

The build output will be stored in the `/dist` directory.
